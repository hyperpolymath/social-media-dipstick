# Vulnerability Disclosure

We appreciate responsible disclosure of security vulnerabilities.
Please allow up to 90 days for patching before public disclosure.

# Scope

This security policy covers:
- All services in services/ directory
- Infrastructure configurations
- Database schemas
- API endpoints
- Authentication systems

Out of scope:
- Third-party dependencies (report to upstream)
- Social engineering attacks
- Physical security

# Process

1. Email security@nuj.org.uk with:
   - Vulnerability description
   - Steps to reproduce
   - Potential impact
   - Suggested fix (optional)

2. We will acknowledge within 48 hours

3. We will provide updates every 7 days

4. We will coordinate disclosure timeline
